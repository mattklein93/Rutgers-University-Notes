Equivalence

  • Two formulas, F1 ≡ F2 if their truth values are equivalent
    De Morgan's Rule
  • $${\lnot(P \land Q) \equiv \lnot P \lor \lnot Q}$$
  • $${\lnot(P \lor Q) \equiv \lnot P \land \lnot Q}$$
  • $${P \to Q \equiv \lnot P \lor Q}$$

    This is the same because if P is false, then !P is true, then Q could be either true or false, it does not matter. Only one or the other matters as it is an OR statement. If !P is false, Q could still be true.
